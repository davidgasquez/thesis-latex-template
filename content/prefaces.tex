\begin{abstract}
    This thesis examines a solution to the Airbnb machine learning problem
    proposed in the Kaggle Data Science platform. The goal is to predict in
    which country a new user will make his or her first booking.
    The scope of the research presented here is the application of boosting
    algorithms to predict where the new user will book, taking into account the
    performance of such prediction as it need to be fast.
    The project relies in Python programming language and uses diverse Open
    Source libraries such as Scikit-Learn and Pandas.
\end{abstract}
